# akashic-pdi

Akashicのプラットフォーム依存層(Platform Dependent Implementation (PDI) Layer)のインターフェース定義を与えます。

## インストール

```
npm install @akashic/akashic-pdi
```

## 利用方法

```
import * as pdi from "@akashic/akashic-pdi";
```

## 開発

ビルド:

```
npm run build
```

テスト:

```
npm test
```
